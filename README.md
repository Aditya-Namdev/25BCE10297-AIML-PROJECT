#  AI Spending Tracker (Rule-Based Expert System)

##  Project Description
This project is a simple AI-based expense tracking system that helps users monitor their daily spending and receive intelligent suggestions.

The system uses a **rule-based approach (no machine learning)** to categorize expenses and provide advice based on spending patterns and budget limits.

---

## Problem Statement
Many students spend money daily without tracking where it goes. This leads to overspending and poor financial habits.

This project solves that problem by:
- Automatically categorizing expenses
- Monitoring daily spending
- Giving smart suggestions using AI logic

---

## AI Concepts Used
- Rule-Based System
- Pattern Matching (keyword detection)
- Decision Making using IF-ELSE logic

---

## Features
- Categorizes expenses (Food, Travel, Academic, Personal)
- Tracks total daily spending
- Checks against a fixed daily budget
- Gives intelligent suggestions like:
  - Overspending warning
  - Budget alerts
  - Smart saving advice

## How to Use

1. Enter expense description (example: "zomato order", "bus ticket")
2. Enter amount
3. The system will:
   - Detect category automatically
   - Analyze your spending
   - Give AI-based advice

### Example:
Enter expense description: zomato order
Enter amount: 300

Category: Food
Advice: High food expense detected. Consider the hostel mess for your next meal.
Total spent today: 300


---

## AI Logic Explanation
The system works using predefined rules:

- If spending exceeds daily budget → show critical warning  
- If food expense is high → suggest reducing food spending  
- If spending is low → give positive feedback
